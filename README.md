# Marketplace System

A full-stack marketplace web application built with:

- Laravel
- Livewire
- Tailwind CSS
- MySQL
- Orchid (Admin Panel)

## Features

- User roles (Admin, Vendor, Customer, Delivery)
- Vendor product management
- Product categories
- Order system with multiple items
- Payment system
- Delivery tracking
- Product reviews
- Vendor payouts

## Database Design

Includes:

- Users
- Vendors
- Customers
- Products
- Orders & Order Items
- Payments
- Deliveries
- Reviews
- Vendor Payouts

## Setup

```bash
git clone <repo>
cd marketplace
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
